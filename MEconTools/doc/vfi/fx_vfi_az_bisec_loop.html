FF_VFI_AZ_BISEC_LOOP Savings Loop Exact (FOC) Examples

back to Fan's Intro Math for Econ, Matlab Examples, or Dynamic Asset Repositories
This is the example vignette for function: ff_vfi_az_bisec_loop from the MEconTools Package. This function solves the dynamic programming problem for a (a,z) model. Households can save a, and face AR(1) shock z. The problem is solved over the infinite horizon.
This is the looped code, it is slow for larger state-space problems.
The code uses continuous choices, solved with bisection. The state-space is on a grid, but choice grids are in terms of percentage of resources to save and solved exactly.
